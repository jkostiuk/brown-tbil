Problem A.2.2.

This problem asks you to expand on the ideas in Activity 5.2.14.

(a)

Explain and demonstrate how to use integration by parts to show that \(\displaystyle \int\sin^2{x}dx=\frac{1}{2}(-\sin{x}\cos{x}+x)\text{.}\)

(b)

Explain and demonstrate how to use a similar method to evaluate \(\displaystyle \int\cos^2{x}dx\text{.}\)

(c)

More generally, let \(I_n=\displaystyle\int\sin^n{x}dx\text{,}\) where \(n\) a positive integer. Use integration by parts to explain why
\begin{equation*} I_n=\frac{-\sin^{n-1}{x}\cos{x}+(n-1)I_{n-2}}{n}. \end{equation*}
While you do not need to turn it in, consider finding an analogous formula relating the integrals \(\int\cos^n{x}dx\text{.}\)
Hint.
Follow the suggestions in Activity 5.2.14 and be prepared to do some trial-and-error with your integration-by-parts. Consider borrowing an idea from Activity 5.2.12
Solution.

(a)

We begin by using integration by parts. For this, I chose \(u=\sin{x}\) and \(dv=\sin{x}dx\text{.}\) With these choices, we have \(du=\cos{x}dx\text{,}\) \(v=-\cos{x}\text{,}\) and applying the integration by parts formula yields:
\begin{equation*} I=\displaystyle\int\sin^2{x}dx=\int\sin{x}\sin{x}dx=-\sin{x}\cos{x}+\int\cos^2{x}dx. \end{equation*}
Inspired by Activity 5.2.14, we use the identity \(\cos^2{x}=1-\sin^2{x}\) to re-write the above equation as:
\begin{align*} I \amp =\amp \sin{x}\cos{x}+\int dx-\int\sin^2{x}dx\\ \amp =\amp -\sin{x}\cos{x}+x-I. \end{align*}
Moving \(I\) to the left hand side allows us to solve:
\begin{equation*} I=\frac{x-\sin{x}\cos{x}}{2} \end{equation*}
as desired.

(b)

We imitate and modify the techniques we used above. Letting \(I=\int\cos^2{x}dx\text{,}\) we use integration by parts as follows:
\begin{align*} I \amp =\amp \int\cos{x}\cos{x}dx \\ \amp\amp u=\cos{x},\ \ dv=\cos{x}dx \\ \amp\amp du=-\sin{x}dx,\ \ v=\sin{x} \\ \amp =\amp \cos{x}\sin{x}+\int\sin^2{x}dx\\ \amp =\amp \cos{x}\sin{x}+\int dx-\int\cos^2{x}dx\\ \amp =\amp \cos{x}\sin{x}+x-I, \end{align*}
where, in the second last line, we used the identity \(\sin^2{x}=1-\cos^2{x}\text{.}\)
From here, we solve for \(I\text{:}\)
\begin{equation*} I=\frac{x+\cos{x}\sin{x}}{2}. \end{equation*}

(c)

While the algebra might look a little weird, let’s see if we can use what we’ve done to help us here. Start with \(I_n=\int\sin^n{x}dx\text{.}\) If we let \(u=\sin^{n-1}{x}dx\) and \(dv=\sin{x}dx\) and apply the integration by parts formula, we get:
\begin{equation*} I_n=-\sin^{n-1}{x}\cos{x}+\int(n-1)\sin^{n-2}{x}\cos^2{x}dx. \end{equation*}
Using the identity \(\cos^2{x}=1-\sin^2{x}\text{,}\) we can continue as:
\begin{align*} I \amp =\amp -\sin^{n-1}{x}\cos{x}+\displaystyle\int (n-1)\sin^{n-2}{x}(1-\sin^2{x})dx\\ \amp =\amp -\sin^{n-1}{x}\cos{x}+(n-1)\int\sin^{n-2}{x}dx-(n-1)\int\sin^n{x}dx\\ \amp =\amp \sin^{n-1}{x}\cos{x}+(n-1)I_{n-2}-(n-1)I_n, \end{align*}
from which we can move the \(I_n\) terms to the left to find:
\begin{equation*} nI_n=-\sin^{n-1}{x}\cos{x}+(n-1)I_{n-2}. \end{equation*}
