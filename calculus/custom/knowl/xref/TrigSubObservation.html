Observation 5.4.4.

The identity \(\sin^2\theta+\cos^2\theta=1\) can always be used to help us re-write expressions of the form \(b^2-a^2x^2\) as perfect squares:
\begin{align*} \text{If } x \amp =\frac{b}{a}\sin\theta\\ \text{then } b^2-a^2x^2\amp =b^2-b^2\sin^2\theta =b^2\cos^2\theta. \end{align*}
More generally, the identities \(1+\tan^2\theta=\sec^2\theta\) and \(\sec^2\theta-1=\tan^2\theta\) help us re-write the following expression as perfect squares:
\begin{align*} \text{If } x \amp =\frac{b}{a}\tan\theta\\ \text{then } b^2+a^2x^2\amp =b^2+b^2\tan^2\theta =b^2\sec^2\theta. \end{align*}
\begin{align*} \text{If } x \amp =\frac{b}{a}\sec\theta\\ \text{then } a^2x^2-b^2\amp =b^2\sec^2\theta-b^2=b^2\tan^2\theta. \end{align*}
A substitution of this kind is called a trigonometric substitution.
